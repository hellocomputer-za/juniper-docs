# Juniper UI Documentation

**Project Version**: 2.0.0  
**Last Updated**: 2024-12-19

## Description

Juniper UI is a comprehensive design system and component library built for ApostropheCMS 4.x applications. It provides a robust foundation of reusable components, utilities, and patterns that enable rapid development of modern, accessible, and responsive web applications.

### Installation Instructions

#### 1. Clone the Repository
```bash
git clone https://github.com/your-org/juniper-ui.git
cd juniper-ui
```

#### 2. Install Dependencies
```bash
npm install
```

#### 3. Environment Setup
```bash
# Copy environment file
cp .env.example .env

# Set your theme
export THEME=juniper
```

#### 4. Build Assets
```bash
# Development
npm run dev

# Production
npm run build
```

#### 5. Start Development Server
```bash
npm start
```

## Usage Examples

### Basic Implementation

#### 1. Include Required Libraries
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Juniper UI Example</title>
    
    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
    
    <!-- Swiper CSS -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.css">
    
    <!-- AOS CSS -->
    <link href="https://unpkg.com/aos@2.3.1/dist/aos.css" rel="stylesheet">
    
    <!-- Bootstrap Icons -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.2/font/bootstrap-icons.css">
    
    <!-- Juniper UI CSS -->
    <link rel="stylesheet" href="/dist/css/juniper-ui.min.css">
</head>
<body>
    <!-- Your content here -->
    
    <!-- Bootstrap JS -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
    
    <!-- Swiper JS -->
    <script src="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.js"></script>
    
    <!-- AOS JS -->
    <script src="https://unpkg.com/aos@2.3.1/dist/aos.js"></script>
    
    <!-- Juniper UI JS -->
    <script src="/dist/js/juniper-ui.min.js"></script>
</body>
</html>
```

#### 2. Initialize Components
```javascript
// Initialize AOS
AOS.init({
    duration: 800,
    easing: 'ease-out',
    once: true
});

// Initialize Juniper UI components
document.addEventListener('DOMContentLoaded', function() {
    // Initialize all Juniper components
    JuniperUI.init();
});
```

### Advanced Configurations

#### 1. Custom Theme Configuration
```scss
// _custom-theme.scss
$juniper-primary: #007bff;
$juniper-secondary: #6c757d;
$juniper-success: #28a745;
$juniper-danger: #dc3545;
$juniper-warning: #ffc107;
$juniper-info: #17a2b8;

// Import Juniper UI with custom variables

```

#### 2. Component-Specific Configuration
```javascript
// Custom component configuration
const juniperConfig = {
    accordion: {
        animation: 'slide',
        duration: 300,
        easing: 'ease-out'
    },
    slider: {
        autoplay: true,
        delay: 5000,
        effect: 'fade'

    animations: {
        enabled: true,
        duration: 800,
        offset: 100
```

```scss
    xxl: 1400px

    }
}
```

## Troubleshooting

### Common Issues and Solutions

#### 1. Components Not Initializing
**Problem**: Juniper UI components not working
**Solution**:
```javascript
// Check if libraries are loaded
if (typeof Bootstrap !== 'undefined' && typeof Swiper !== 'undefined') {
    // Initialize components
    JuniperUI.init();
} else {
    console.error('Required libraries not loaded');
}
```

#### 2. CSS Not Loading
**Problem**: Styles not applying correctly
**Solution**:
```html
<!-- Ensure proper CSS loading order -->
<link href="bootstrap.min.css" rel="stylesheet">
<link href="juniper-ui.min.css" rel="stylesheet">
```

#### 3. JavaScript Errors
**Problem**: Console errors related to Juniper UI
**Solution**:
```javascript
// Enable debug mode
JuniperUI.init({
    debug: true,
    logLevel: 'verbose'
});
```

#### 4. Responsive Issues
**Problem**: Components not responsive on mobile
**Solution**:
```scss
// Ensure proper viewport meta tag
<meta name="viewport" content="width=device-width, initial-scale=1.0">

// Check CSS media queries
@media (max-width: 768px) {
    .component {
        // Mobile-specific styles
    }
}
```

#### 5. Performance Issues
**Problem**: Slow loading or rendering
**Solution**:
```javascript
// Optimize component loading
JuniperUI.init({
    lazyLoad: true,
    preloadImages: false,
    optimizeAnimations: true
});
```

```javascript
// Enable comprehensive debugging
const debugConfig = {
```

```javascript
// Monitor component initialization
JuniperUI.on('init', function(component) {

JuniperUI.on('error', function(error) {
    console.error('Juniper UI Error:', error);
```

#### 3. Performance Monitoring
```javascript
// Monitor performance metrics
const startTime = performance.now();

JuniperUI.on('ready', function() {
```

## Performance Tips

### 1. Asset Optimization
```bash
# Optimize CSS
npm run build:css -- --minify

# Optimize JavaScript
npm run build:js -- --minify

# Optimize images
npm run optimize:images
```

### 2. Lazy Loading
```javascript
// Enable lazy loading for components
JuniperUI.init({
    lazyLoad: {
        components: true,
        images: true,
        threshold: 0.1
    }
});
```

### 3. Bundle Optimization
```javascript
// Tree shaking for unused components
import { Button, Accordion, Slider } from 'juniper-ui';

// Only import what you need
const components = [Button, Accordion];
JuniperUI.init({ components });
```

### 4. Caching Strategies
```html
<!-- Set proper cache headers -->
<meta http-equiv="Cache-Control" content="max-age=31536000">
<link rel="preload" href="/dist/css/juniper-ui.min.css" as="style">
<link rel="preload" href="/dist/js/juniper-ui.min.js" as="script">
```

### 5. Critical CSS
```html
<!-- Inline critical CSS -->
<style>
    /* Critical Juniper UI styles */
    .header-slider-widget { height: 100vh; }
    .accordion-widget { margin: 1rem 0; }
    .button-widget { display: flex; gap: 1rem; }
</style>
```

### 6. Image Optimization
```html
<!-- Use responsive images -->
<img src="image.jpg" 
     srcset="image-300w.jpg 300w, image-600w.jpg 600w, image-900w.jpg 900w"
     sizes="(max-width: 768px) 100vw, 50vw"
     alt="Description">
```

### 7. JavaScript Performance
```javascript
// Debounce event handlers
const debouncedHandler = JuniperUI.utils.debounce(function() {
    // Handle event
}, 250);

// Use requestAnimationFrame for animations
JuniperUI.utils.raf(function() {
    // Animation code
});
```

### 8. Memory Management
```javascript
// Cleanup on page unload
window.addEventListener('beforeunload', function() {
    JuniperUI.destroy();
});

// Cleanup specific components
JuniperUI.destroyComponent('slider');
```

### 9. CDN Optimization
```html
<!-- Use CDN with fallback -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
<script>
    if (typeof Bootstrap === 'undefined') {
        document.write('<script src="/local/bootstrap.min.js"><\/script>');
    }
</script>
```

### 10. Monitoring and Analytics
```javascript
// Performance monitoring
JuniperUI.monitor({
    metrics: ['loadTime', 'renderTime', 'interactionTime'],
    report: function(data) {
        // Send to analytics
        analytics.track('juniper_performance', data);
    }
});
```
