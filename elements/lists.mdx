# Lists Component

## Description

A comprehensive list system for ordered and unordered lists built with Bootstrap framework. This component provides semantic list elements with enhanced styling options, including custom bullet points, numbering systems, and interactive states. The lists use Bootstrap's utility classes and can be easily customized with animations, spacing, and advanced styling options through the Juniper UI system.

## HTML Structure

The lists component uses semantic HTML elements with Bootstrap classes and optional AOS animations:

```html
<!-- Unordered Lists -->
<ul class="list-unstyled">
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

<ul class="list-inline">
  <li class="list-inline-item">Item 1</li>
  <li class="list-inline-item">Item 2</li>
  <li class="list-inline-item">Item 3</li>
</ul>

<!-- Ordered Lists -->
<ol class="list-decimal">
  <li>First ordered item</li>
  <li>Second ordered item</li>
  <li>Third ordered item</li>
</ol>

<!-- Description Lists -->
<dl class="row">
  <dt class="col-sm-3">Term</dt>
  <dd class="col-sm-9">Description for the term</dd>
  <dt class="col-sm-3">Another term</dt>
  <dd class="col-sm-9">Description for another term</dd>
</dl>
```

### Nunjucks Template Implementation

When using with dynamic content and animations:

```nunjucks
{% import "@juniper/ui/animations.html" as animations %}

<!-- Animated List -->
<div {{ animations.el(data.widget) }}>
  <ul class="list-custom">
    {% for item in data.listItems %}
      <li class="list-item-animated" 
          data-aos="fade-up" 
          data-aos-delay="{{ loop.index0 * 100 }}">
        {{ item.text }}
      </li>
    {% endfor %}
  </ul>
</div>

<!-- Link List Integration -->
{% import "@juniper/ui/anchor.html" as anchor %}

<ul class="list-links">
  {% for link in data.links %}
    <li>
      {% render anchor.regular(link, "list-link-item") %}
    </li>
  {% endfor %}
</ul>
```

### List Group Structure (Bootstrap Components)

```html
<!-- Basic List Group -->
<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action active">
    Active list item
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    Second list item
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    Third list item
  </a>
</div>

<!-- List Group with Content -->
<div class="list-group">
  <div class="list-group-item list-group-item-action">
    <div class="d-flex w-100 justify-content-between">
      <h5 class="mb-1">List group item heading</h5>
      <small>3 days ago</small>
    </div>
    <p class="mb-1">Some placeholder content in a paragraph.</p>
    <small>And some small print.</small>
  </div>
</div>
```

### HTML Attributes Explanation

- `{{ animations.el(data.widget) }}`: Juniper UI animation element helper
- `list-unstyled`: Removes default list styling
- `list-inline`: Creates horizontal list layout
- `list-inline-item`: Individual inline list items
- `list-group`: Bootstrap component for interactive lists
- `list-group-item`: Individual list group items
- `list-group-item-action`: Adds hover/focus states
- `active`: Marks the current/active list item

## Animations

The lists component utilizes AOS (Animate On Scroll) library for scroll-triggered animations:

### Nunjucks Animation Template Implementation

```nunjucks
{% import "@juniper/ui/animations.html" as animations %}

<!-- Animated list with staggered items -->
<div {{ animations.el({
  aos: "fade-up",
  duration: 600
}) }}>
  <ul class="list-custom">
    {% for item in data.items %}
      <li data-aos="slide-right" 
          data-aos-delay="{{ loop.index0 * 150 }}"
          data-aos-duration="500">
        {{ item.text }}
      </li>
    {% endfor %}
  </ul>
</div>

<!-- List group with animation -->
<div class="list-group" 
     data-aos="zoom-in" 
     data-aos-duration="800">
  {% for item in data.groupItems %}
    <div class="list-group-item" 
         data-aos="fade-left" 
         data-aos-delay="{{ loop.index0 * 100 }}">
      {{ item.content }}
    </div>
  {% endfor %}
</div>
```

### Available AOS Animation Effects

#### Fade animations:
- `fade`
- `fade-up`
- `fade-down`
- `fade-left`
- `fade-right`

#### Slide animations:
- `slide-up`
- `slide-down`
- `slide-left`
- `slide-right`

#### Zoom animations:
- `zoom-in`
- `zoom-in-up`
- `zoom-in-down`
- `zoom-in-left`
- `zoom-in-right`
